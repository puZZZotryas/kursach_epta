STasks - Web-приложение для мониторинга успеваемости студентов

📋 О проекте

STasks — это современное web-приложение, разработанное для эффективного мониторинга успеваемости студентов. Проект представляет собой полностью реализованную клиентскую часть приложения с продуманным пользовательским интерфейсом и интерактивными элементами.

🎯 Цель разработки

Создание интуитивно понятного, визуально привлекательного и функционального интерфейса для взаимодействия между студентами, преподавателями и администрацией образовательного учреждения.

🔑 Ключевые особенности

- 👥 Ролевая модель: Разделение функционала для студентов, преподавателей и администраторов
- 🎨 Современный UI/UX: Минималистичный дизайн с четкой визуальной иерархией
- 📱 Адаптивный дизайн: Комфортное использование на различных устройствах
- 🎯 Интерактивность: Динамическое взаимодействие без перезагрузки страниц
- 🎨 Дизайн-система: Единая библиотека компонентов для согласованного интерфейса

🛠 Технологии и инструменты

Дизайн и прототипирование
- Figma — создание дизайн-системы, UI-кита и интерактивного прототипа
- UI-кит — библиотека переиспользуемых компонентов

Клиентская часть
- HTML5 — семантическая структура страниц
- CSS3 — стилизация, адаптивная верстка, CSS-переменные
- JavaScript (ES6+) — интерактивность, валидация форм, AJAX-запросы

Принципы разработки
- User-Centered Design — подход, ориентированный на пользователя
- Компонентный подход — переиспользуемые элементы интерфейса
- Семантическая верстка — доступность и SEO-оптимизация

### UI-компоненты

- Кнопки (Primary, Secondary, Ghost, Danger)
- Поля ввода с валидацией
- Карточки курсов и заданий
- Модальные окна
- Таблицы расписания
- Индикаторы прогресса

🚀 Функциональные возможности

Для студентов
- 📊 Просмотр дашборда с успеваемостью
- 📅 Календарь занятий и дедлайнов
- 📈 Графики академического прогресса
- 📝 История оценок по семестрам

Для преподавателей
- ✏️ Создание и редактирование заданий
- 📋 Выставление оценок (индивидуально и массово)
- 📊 Аналитика по группе
- 🔔 Отправка уведомлений студентам

Для администраторов
- 👥 Управление пользователями (CRUD)
- 📚 Настройка курсов и групп
- ⏰ Редактирование расписания (Drag-and-drop)
- 🛡️ Настройка прав доступа
